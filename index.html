Compaginator

A framework agnostic JavaScript pagination component written in CoffeeScript.

Download this project as a .zip file Download this project as a tar.gz file

Philosophy

Compaginator separates presentation from business logic in order to provide flexibility. Exposing a single model that encapsulates the common pagination business logic, and then providing library/framework specific components for easy integration into existing code bases.

You should familiarise yourself with the core compaginator model, and then read the documentation relating to the component for your target library/framework.

Installation

Include the compaginator.min.js or compaginator.js file on your page. 

<script type="text/javascript" src="compaginator.min.js"></script>

Compaginator.Model

This class represents the core compaginator model.

Example Usage (CoffeeScript)

paginator = new Compaginator.Model()
paginator.setNumPages 100
paginator.setNumPagesDisplayed 10
paginator.setNumPagesAnchored 2
paginator.setCurrentPage 10
pages = paginator.getDisplayPages()

# logs [1, 2, ..., 8, 9, 10, 11, 12, ..., 99, 100]
console.log pages.map (page) -> return page.page

The following methods form the public API for this class.

Method Description
setCurrentPage(number) Set the current page of the paginator.
setNextPage() Set the next page as the current page.
setPrevPage() Set the prev page as the current page.
setNumPagesDisplayed(number) Set the number of pages to be displayed when rendered.
setNumPagesAnchored(number) Set the number of pages to anchor at the start & end.
setNumPages(number) Set the total number of pages to paginate.
getCurrentPage() Returns the current page number.
getDisplayPages() Returns the subset of pages to render in the view.
observe(event, callback, [context]) Observes the specified event and calls the callback whenever the event occurs.

Events

The following events can be passed to the obseve() method

Event Description
page-changed Triggered whenever the current page is changed.

jQuery

For the full example please refer to the GitHub repo.

Example Usage (CoffeeScript)

$(document).ready ->

  paginator = new Compaginator.Model()
  paginator.setNumPages 20
  paginator.observe 'page-changed', ->
    topPaginatorView.render paginator, '.pagination-top'
    bottomPaginatorView.render paginator, '.pagination-bottom'

  topPaginatorView = new Compaginator.JqueryView paginator, '.pagination-top'
  bottomPaginatorView = new Compaginator.JqueryView paginator, '.pagination-bottom'
