Welcome to Compaginator

A framework agnostic JavaScript pagination component written in CoffeeScript.

Philosophy

Compaginator separates presentation from business logic in order to provide flexibility. This is achieved by creating a single model that encapsulates the common pagination business logic, and then providing library/framework specific views for easy integration into existing code bases - regardless of underlying architecture.

This means that you should familiarise yourself with the core compaginator model, and then read the documentation relating to the view for your target library/framework.

Installation

Include the compaginator.min.js or compaginator.js file on your page. 

<script type="text/javascript" src="compaginator.min.js"></script>

Compaginator.Model

This class represents the core compaginator model. The following methods form the public API for this class.

Method Description
setCurrentPage(number) Set the current page of the paginator.
setNextPage() Set the next page as the current page.
setPrevPage() Set the prev page as the current page.
setNumPagesDisplayed(number) Set the number of pages to be displayed when rendered.
setNumPagesAnchored(number) Set the number of pages to anchor at the start & end.
setNumPages(number) Set the total number of pages to paginate.
getCurrentPage() Returns the current page number.
getDisplayPages() Returns the subset of pages to render in the view.
